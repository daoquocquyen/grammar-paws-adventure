## AI-Agent Usage Rules
- Read this hub first, then open only the minimum linked files needed for the task.
- Treat planning artifacts as requirements source, and `src/ui/stitch` as implemented-state source.
- If data is missing, record `TBD` explicitly; do not invent.
- Keep updates modular (edit specific doc file, avoid monolithic context docs).

## Update Trigger
Update relevant file(s) in this hub when any of these change:
- MVP scope or feature boundaries
- screen routing/files
- coding/security/testing/delivery standards
- UX behavior across implemented screens
